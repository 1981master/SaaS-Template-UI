1️⃣ Layout & Navigation
Tabs – switch between sections without reloading content.
Accordion / Collapse – expandable panels.
Breadcrumbs – show navigation hierarchy.
Sidebar – collapsible navigation panel.
Navbar / Header – top navigation with logo, links, and actions.
Pagination – for lists or tables.
2️⃣ Inputs & Forms
Slider / Range Input – select numeric values.
Toggle / Switch – for boolean settings (you already have).
Select / Dropdown – single or multi-select.
Autocomplete / Typeahead – suggestions as user types.
Rating / Stars – for feedback inputs.
Tag / Chip Input – add/remove tags or labels.
3️⃣ Feedback & Status
Alert / Banner – messages at top of page.
Notification / Toast – temporary messages.
Loader / Skeletons – show while fetching data.
Progress / Steps – indicate progress in a flow.
4️⃣ Display & Visualization
Card / Tile – content containers with shadow.
Avatar / Badge – small profile image or status indicator.
Tooltip / Popover – extra info on hover/click (you have tooltip).
Table / Data Grid – structured data display.
Tag / Badge – highlight categories or statuses.
5️⃣ Media & Interaction
Carousel / Slider – images or content slides.
Image / Video Viewer – lightbox style.
File Upload / Drag & Drop – already done.
Modal / Drawer / Offcanvas – you have modal, drawer is nice.
6️⃣ Utility / Misc
Divider / Separator – horizontal or vertical.
Spacer / Stack – consistent spacing in layouts.
Copy to Clipboard – small button to copy text.
Skeleton Loader – visually show loading structure.
Empty State – message when no content exists.
